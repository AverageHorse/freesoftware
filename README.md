ТРЕЙ

■ Battery Mode: https://bmode.tarcode.ru/ - приложение, которое заменяет стандартный индикатор батареи Windows на более красивый и функциональный, а также добавляет удобное управление подсветкой.

■ EarTrumpet: https://eartrumpet.app/ - заменяет и расширяет возможности управления звуком из трея. Управляет громкостью классических и современных приложений, управляет аудио устройством по-умолчанию в два клика, поддерживает перемещение приложений между устройствами воспроизведения.

■ MiniBin: https://github.com/kobaltgit/minibin - иконка в системном трее для удобной очистки корзины с возможностью автоматизации процесса.

■ Ditto - Clipboard Manager: https://sabrogden.github.io/Ditto/ - программа является расширением стандартного буфера обмена Windows. Она сохраняет каждый элемент, помещенный в буфер обмена, позволяя вам получить доступ к любому из этих элементов позднее. Также позволяет сохранять любой тип информации, которая может быть помещена в буфер обмена, текст, изображения, HTML, пользовательские форматы и др.

■ John’s Background Switcher: https://johnsad.ventures/software/backgroundswitcher/windows/ - программа, периодически меняющая обои на вашем Рабочем столе. Вы можете выбрать отдельные фотографии, картинки, а так же целые папки с компьютера, или из множества сетевых источников - Flickr, Facebook, Instagram и т.д.

■ PassMark RAMMon: passmark.com/products/rammon/ - утилита для получения и отображения информации SPD (Serial Presence Detect) из установленных на ПК модулей оперативной памяти.

■ CPU-Z: https://www.cpuid.com/softwares/cpu-z.html - программа-утилита для отображения технической информации о персональном компьютере пользователя.

■ GPU-Z: https://www.techpowerup.com/gpuz/ - программа для отображения технической информации о видеоадаптере.

■ WinDirStat: https://windirstat.net/ - свободная бесплатная программа для анализа использования ёмкости разделов диска.

■ LockHunter: https://lockhunter.com/ - инструмент, который может разблокировать файлы и папки, используемые другими программами или процессами.

■ WhySoSlow: https://www.resplendence.com/whysoslow -  это программное обеспечение использует целостный подход к анализу быстродействия и производительности системы и предназначено как универсальное решение для проверки всех показателей работоспособности системы.

СИСТЕМА

■ Context Menu Manager: https://github.com/BluePointLilac/ContextMenuManager - утилита для просмотра и удаления пунктов из контекстных меню.

■ Simple Sticky Notes: https://www.simplestickynotes.com/ - простая в понимании программа и более продвинутая альтернатива «Заметкам Windows».

■ Far Manager: https://www.farmanager.com/ - консольный файловый менеджер, предоставляет удобный интерфейс пользователя для работы с файловыми системами (реальными и эмулированными) и файлами.

■ Everything: https://www.voidtools.com/ - мгновенный поиск файлов и папок по их именам.

■ LocalSend: https://localsend.org/ - бесплатный кроссплатформенный инструмент для обмена файлами с открытым исходным кодом, который позволяет вам обмениваться файлами с близлежащими устройствами.

■ Monarch: https://github.com/Nuzair46/Monarch - отсоединяйте, восстанавливайте и переключайте расположение мониторов, не прикасаясь к кабелям.

■ Ext2Fsd: https://sourceforge.net/projects/ext2fsd/ - драйвер файловой системы Linux ext2/ext3 с открытым исходным кодом для систем Windows (2K/XP/VISTA/WIN7/WIN8, X86/AMD64).

■ OpenRGB: https://openrgb.org/ - набор программного обеспечения, который позволяет управлять всеми вашими RGB-устройствами.

■ Ventoy: https://www.ventoy.net/ - инструмент с открытым исходным кодом для создания загрузочных USB-накопителей с поддержкой ISO/WIM/IMG/VHD(x)/EFI.
