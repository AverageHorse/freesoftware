----
ТРЕЙ

■ Battery Mode: https://bmode.tarcode.ru/ - приложение, которое заменяет стандартный индикатор батареи Windows на более красивый и функциональный, а также добавляет удобное управление подсветкой.

■ EarTrumpet: https://eartrumpet.app/ - заменяет и расширяет возможности управления звуком из трея. Управляет громкостью классических и современных приложений, управляет аудио устройством по-умолчанию в два клика, поддерживает перемещение приложений между устройствами воспроизведения.

■ MiniBin: https://github.com/kobaltgit/minibin - иконка в системном трее для удобной очистки корзины с возможностью автоматизации процесса.

■ Ditto - Clipboard Manager: https://sabrogden.github.io/Ditto/ - программа является расширением стандартного буфера обмена Windows. Она сохраняет каждый элемент, помещенный в буфер обмена, позволяя вам получить доступ к любому из этих элементов позднее. Также позволяет сохранять любой тип информации, которая может быть помещена в буфер обмена, текст, изображения, HTML, пользовательские форматы и др.

■ John’s Background Switcher: https://johnsad.ventures/software/backgroundswitcher/windows/ - программа периодически меняющая обои на вашем Рабочем столе. Вы можете выбрать отдельные фотографии, картинки, а так же целые папки с компьютера, или из множества сетевых источников - Flickr, Facebook, Instagram и т.д.

ИНФОРМАЦИЯ

■ Open Hardware Monitor: https://openhardwaremonitor.org/ - программное обеспечение с открытым исходным кодом, которое отслеживает датчики температуры, частоту вращения вентиляторов, напряжение, нагрузку и тактовую частоту компьютера.

■ GPU Shark II: https://www.geeks3d.com/gpushark/ - это информационно-мониторинговая утилита для видеокарт.

■ CPU-Z: https://www.cpuid.com/softwares/cpu-z.html - программа-утилита для отображения технической информации о персональном компьютере пользователя.

■ GPU-Z: https://www.techpowerup.com/gpuz/ - программа для отображения технической информации о видеоадаптере.

■ WinDirStat: https://windirstat.net/ - свободная бесплатная программа для анализа использования ёмкости разделов диска.

■ LockHunter: https://lockhunter.com/ - инструмент, который может разблокировать файлы и папки, используемые другими программами или процессами.

■ WhySoSlow: https://www.resplendence.com/whysoslow -  это программное обеспечение использует целостный подход к анализу быстродействия и производительности системы и предназначено как универсальное решение для проверки всех показателей работоспособности системы.

СИСТЕМА

■ O&O ShutUp10++: https://www.oo-software.com/en/shutup10 - антишпион для Windows 10/11.

■ HostsFileEditor: https://hostsfileeditor.com/ - редактирование файла hosts для Windows и управление им.

■ Geek Uninstaller: https://geekuninstaller.com/ - портативная утилита, которая выполняет глубокое и быстрое сканирование и удаляет все остатки удаленных программ. Также поддерживает удаление приложений Microsoft Store и имеет встроенную поддержку X64.

■ Delete.On.Reboot: https://www.softwareok.com/?seite=Freeware/Delete.On.Reboot - небольшая и эффективная бесплатная программа, позволяющая элегантно удалять нестираемые файлы, каталоги, вложенные папки при перезагрузке компьютера под управлением Windows.

■ Far Manager: https://www.farmanager.com/ - консольный файловый менеджер, предоставляет удобный интерфейс пользователя для работы с файловыми системами (реальными и эмулированными) и файлами.

■ Everything: https://www.voidtools.com/ - мгновенный поиск файлов и папок по их именам.

■ LocalSend: https://localsend.org/ - бесплатный кроссплатформенный инструмент для обмена файлами с открытым исходным кодом, который позволяет вам обмениваться файлами с близлежащими устройствами.

■ Monarch: https://github.com/Nuzair46/Monarch - отсоединяйте, восстанавливайте и переключайте расположение мониторов, не прикасаясь к кабелям.

■ DefenderUI: https://www.defenderui.com/ - бесплатная программа, являющаяся альтернативной графической оболочкой для настройки Windows Defender.

■ SimpleWall: https://github.com/henrypp/simplewall - простой инструмент для настройки платформы фильтрации Windows (WFP), который может настраивать сетевую активность на вашем компьютере. Не является надстройкой для встроенного системного файрвола, работает отдельно.

■ Mem Reduct: https://github.com/henrypp/memreduct - легкое приложение для управления памятью в режиме реального времени для мониторинга и очистки системной памяти вашего компьютера.

■ Victoria: https://hdd.by/victoria/ - программа для диагностики, исследования, тестирования и мелкого ремонта жёстких дисков, SSD-накопителей, карт памяти, а также любых других накопителей.

■ Ext2Fsd: https://sourceforge.net/projects/ext2fsd/ - драйвер файловой системы Linux ext2/ext3 с открытым исходным кодом для систем Windows (2K/XP/VISTA/WIN7/WIN8, X86/AMD64).

■ OpenRGB: https://openrgb.org/ - набор программного обеспечения, который позволяет управлять всеми вашими RGB-устройствами.

■ Ventoy: https://www.ventoy.net/ - инструмент с открытым исходным кодом для создания загрузочных USB-накопителей с поддержкой ISO/WIM/IMG/VHD(x)/EFI.

РАЗГОН

■ QuickCPU: https://coderbag.com/product/quickcpu - приложение, разработанное для точной настройки и мониторинга производительности процессора, параметров питания и напряжения, таких как: парковка ядер, масштабирование частоты, Turbo Boost, C-States, Speed Shift и FIVR Контроль, а также некоторые другие возможности.

ГРАФИКА

■ Krita: https://krita.org/ - профессиональный бесплатный растровый графический редактор как для любителей, так и для профессиональных художников и иллюстраторов, использующих графические планшеты в процессе создания цифровой живописи как в домашних условиях, так и в игровых студиях, в VFX мастерских и дизайн-студиях.

■ GIMP: https://www.gimp.org/ - средство для работы с изображениями, может справиться с задачами создания графики и логотипов, масштабирования и кадрирования фотографий, раскраской, комбинирования изображений с использованием слоев, ретуширования и преобразования изображений в различные форматы.

■ FastStone Image Viewer: https://www.faststone.org/FSViewerDetail.htm - быстрый, стабильный и удобный браузер, конвертер и редактор изображений.

■ ImageGlass: https://imageglass.org/ - легкий и универсальный просмотрщик изображений.

■ Kalmuri: https://global.kilho.net/ru/kalmuri - удобный инструмент для захвата экрана с минимальным набором опций.

ПЛЕЕРЫ

■ Foobar2000: https://www.foobar2000.org/ - аудиоплеер, подкупающий своим весьма качественным звуком, имеющий широкие возможности настройки, спартанский непритязательный интерфейс и довольно низкие системные требования.

■ AIMP: https://aimp.ru/ - мощный, бесплатный аудиоплеер для ПК с поддержкой локальных файлов, подкастов, облачных и сетевых хранилищ, а так же включающий в себя обширный инструментарий для работы с аудиофайлами.
